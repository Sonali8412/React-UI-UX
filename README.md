# Modern UI/UX Website

## 👩‍💻 Developed By
Sonali Gupta

## 📌 Project Description
This is a modern UI/UX web application built using React.  
The project focuses on responsive design, reusable components, and clean UI development.

## 🚀 Features
- Responsive UI design
- Reusable React components
- Modern layout using Flexbox & Grid
- Smooth animations
- Mobile-friendly design

## 🛠 Tech Stack
- React.js
- JavaScript
- HTML5
- CSS3

## ▶️ How to Run Project
1. Clone the repository
2. Install dependencies:
   npm install
3. Start development server:
   npm start
